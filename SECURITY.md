# Security Policy

This project is designed for local spreadsheet processing. It should not upload
data to a remote service.

## Reporting a Vulnerability

Open a private security advisory or contact the maintainer directly if you find
a vulnerability that could expose local files, secrets, or business data.

## Data Handling

Never publish real WMS, ERP, MES, customer, supplier, employee, order, BOM,
shipment, finance, HR, production, procurement, or service exports in this
repository.
