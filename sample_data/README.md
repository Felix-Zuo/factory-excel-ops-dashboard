# Sample Data

These files are generic and safe to share. They model a reusable operations
flow rather than one product category:

- `inventory_sample.csv`: item availability by location.
- `demand_sample.csv`: demand requests by account.
- `fulfillment_sample.csv`: fulfillment progress.
- `replenishment_sample.csv`: replenishment plan lines.
- `work_output_sample.csv`: work output or capacity completion lines.

The files are intentionally small so the sample pipeline can run quickly in CI
and on a new contributor's machine.

Do not replace these files with real WMS, ERP, MES, customer, supplier, BOM,
shipment, production, procurement, service, finance, or HR exports.
