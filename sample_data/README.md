# Sample Data

These files are synthetic and safe to publish. They model a generic operations
flow rather than one private product category:

- `inventory_demo.csv`: item availability by location.
- `demand_demo.csv`: demand requests by account.
- `fulfillment_demo.csv`: fulfillment progress.
- `replenishment_demo.csv`: replenishment plan lines.
- `work_output_demo.csv`: work output or capacity completion lines.

The files are intentionally small so the demo pipeline can run quickly in CI
and on a new contributor's machine.

Do not replace these files with real WMS, ERP, MES, customer, supplier, BOM,
shipment, production, procurement, service, finance, or HR exports.
